# Сервис записи на СТО

## Что это
Сайт, где клиент автосервиса записывается на услугу, и админка,
где владелец видит все записи. О новых записях владельцу приходит
уведомление в Telegram.

## Кто пользуется
- Клиент: хочет записаться за минуту с телефона, без регистрации.
- Владелец: смотрит записи в админке, получает уведомления.

## Экраны
1. Страница записи: название СТО, список услуг, выбор даты и времени,
   поля «Имя», «Телефон», «Марка и модель авто», кнопка «Записаться».
   После отправки — экран «Запись подтверждена» с деталями.
2. Админка по адресу /admin: вход по email и паролю (один владелец).
   Внутри — таблица записей, новые сверху: услуга, дата, время, имя,
   телефон, авто, когда создана.

## Услуги (фиксированный список)
- Замена масла
- Шиномонтаж
- Диагностика подвески
- Замена тормозных колодок
- Развал-схождение
- Компьютерная диагностика

## Данные
Запись: услуга, дата, время, имя клиента, телефон, авто,
дата создания. Хранится в Supabase, таблица bookings.

## Чего НЕ делаем в первой версии
- Никакой оплаты.
- Никакой регистрации и личного кабинета для клиентов.
- Слоты времени не блокируются: два клиента могут выбрать одно время,
  владелец разруливает сам. Улучшим потом.
- Никаких лишних страниц: только запись и админка.

## Технологии
Фронтенд: React (Vite) + Tailwind CSS. Бэкенд и база: Supabase
(таблица + вход для админки). Уведомления в Telegram.
Интерфейс на русском, в первую очередь под телефон.

## Готово, когда
- Клиент с телефона записывается меньше чем за минуту.
- Пустое имя или кривой телефон — понятная ошибка, запись не уходит.
- Запись сразу видна в админке.
- Владельцу приходит сообщение в Telegram с деталями записи.
